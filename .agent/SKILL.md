## Rules for Generation
1. **Selective Referencing:** Only load the reference images for the specific characters mentioned in the prompt.
2. **Individual Identity:** Do not "blend" features. E.g. Keep Ivan's height and Ambrosius's hair color distinct.
3. **Multi-Character Layouts:** If two or more characters are requested, use the reference images to maintain their relative height and scale.
